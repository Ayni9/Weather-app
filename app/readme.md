## Build it yourself

1. get you're api key on weatherapi.com
2. put your key in "API_KEY" string on ./mainactivity.kt
```
API_KEY = <your api key>
``` 
3. Build the app using Android Studio.

> weatherapi URL : `weatherapi.com`
